# Work Note

A Flask-based Task Management Web Application.

## Features

- User Registration
- User Login & Logout
- Password Hashing (Bcrypt)
- Multiple Users
- Add Tasks
- Edit Tasks
- Delete Tasks
- Complete/Incomplete Tasks
- Dashboard Statistics
- Dark/Light Theme
- Responsive Design

## Technologies Used

- Python
- Flask
- SQLite
- SQLAlchemy
- Bcrypt
- HTML
- CSS
- Bootstrap

## Installation

```bash
pip install flask flask_sqlalchemy flask_bcrypt
python app.py
